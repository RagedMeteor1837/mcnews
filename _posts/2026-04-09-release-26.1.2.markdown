Happy Thursday from the shipping room! We're releasing 26.1.2, a hotfix that addresses a couple of critical issues found in the release.<br><br>

**CHANGELOG:**<br>
***Changes:***<br>
\* Checkbox message tooltip is displayed only when text overflows past two rows<br>
\* Confirmation checkbox in Report Player screen now displays tooltip with same text as message<br>
\* Removed baby Herobrine<br>

***Developer's Notes:***<br>
\* We've fixed 1 bug in this release<br>
\* This release is compatible with 26.1 and 26.1.1 servers<br>
\* View the [full changelog on Minecraft.net](https://www.minecraft.net/article/minecraft-java-edition-26-1-2)<br><br>
